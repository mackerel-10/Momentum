## 프로젝트 소개

Chrome 확장 프로그램 Momentum 구현

### 주요기능

- Login
- Clock
- Random Quotes & Background
- To-do list
- Weather(OpenWeatherMap API)
- 사용자 날씨 데이터 API(Query: `lat`, `lon`)

### Setup

```bash
$ git clone https://github.com/mackerel-10/Momentum.git
$ cd Momentum

$ npm install
$ npm start
```

```dotenv
# OpenWeatherMap API Key
API_KEY=
```
